# Todo App

## Overview

Todo App is an Android application developed using Kotlin and Jetpack Compose. The application allows users to manage daily tasks efficiently with Create, Read, Update, and Delete (CRUD) operations. All tasks are stored locally using Room Database, ensuring data persists even after closing the application.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Persistent local storage with Room Database
* Modern user interface built with Jetpack Compose

## Technologies Used

* Kotlin
* Jetpack Compose
* Room Database
* ViewModel
* Repository Pattern
* Android Studio

## Project Structure

* MainActivity
* TodoEntity
* TodoDao
* TodoDatabase
* TodoRepository
* TodoViewModel
* TodoViewModelFactory

## Installation

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle files.
4. Run the application on an emulator or Android device.
